# Validando a Tela de Login

Desafio do curso de React da DIO em que a proposta é criar uma tela de login usando TypeScript e implementar uma etapa de autenticação e validação. Mais especificamente, o desafio pede que o botão de login somente seja ativado quando o usuário preencher os campos de e-mail e senha de maneira adequada.
